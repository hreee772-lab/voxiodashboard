# Voicera Monorepo

This workspace is structured as a monorepo containing the backend service, dashboard, landing page, and call widget codebases.

## Workspace Layout
- `backend/`: FastAPI application containing all core API routes, models, services, and tasks.
- `dashboard/`: Interactive customer support dashboard UI.
- `landing/`: The public marketing website/landing page.
- `widget/`: The interactive support widget demo and code.
